**Fix:** This is typically caused by an outdated ASI loader conflicting with the current game version. ### ❌ Settings not saving between sessions
**Fix:** Run GTA V as Administrator so the mod has write access to save changes in `MenaceMM/settings.ini`.

### ❌ Vehicles or objects not spawning correctly
**Fix:** Confirm the `MenaceMM` config folder is present in the GTA V root directory. Missing config files cause spawner modules to fail silently. Reinstall if the folder is absent.

### ❌ FPS drops during large sessions
**Fix:** Reduce the number of active spawned entities. Open `MenaceMM/settings.ini` and lower `MaxActiveEntities` to a value appropriate for your hardware.
